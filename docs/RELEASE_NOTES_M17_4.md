# Release Notes — Milestone 17 Sprint 17.4

Introduces Team Workspaces and Communities of Practice.

## Included
- Tenant-scoped workspaces with private, internal, and public visibility
- Owner, admin, member, and viewer workspace roles
- Community membership policies and moderator roles
- Workspace announcements with optional expiry
- Polymorphic links to AchieveX resources
- Membership administration safeguards
- REST APIs and administration pages
- Prisma schema and migration

## Validation boundary
Source structure and archive integrity are checked in this package. Dependency installation, Prisma generation, migration execution, compilation, and automated tests remain environment-backed validation steps.
