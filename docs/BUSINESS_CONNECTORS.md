# Business Connectors

Sprint 14.3 adds native adapter contracts for Jira, Azure DevOps, GitHub, GitLab, ServiceNow, Asana, Monday.com and Trello.

The source includes payload normalization, outbound payload shaping, mappings, sync-item history, checksums, conflict records, preview and governed sync APIs. It does not claim live vendor connectivity until provider credentials, OAuth flows, rate-limit handling and network clients are configured in the target environment.
