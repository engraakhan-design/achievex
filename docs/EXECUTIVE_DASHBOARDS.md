# Executive Dashboards
Sprint 13.2 adds role-based dashboards for CEO, COO, CFO, CHRO, PMO and Department Heads. Dashboards are tenant-scoped and composed of reusable widgets backed by versioned metric snapshots.

## Capabilities
- Executive audience templates
- KPI, score, trend, gauge, bar, donut, heatmap and table widget contracts
- Latest value and previous-period change
- Twelve-period time series
- Dimension filtering and drill-down aggregation
- User saved views and default filters
- Immutable metric lineage through Sprint 13.1 snapshots

Templates only reference metric keys when a tenant has not yet created the matching metric definition. This avoids manufacturing analytical values.
