# ADR-036: Enterprise Governance Foundation

## Status
Accepted

## Decision
AchieveX will model governance frameworks, policies, controls, evidence, and assessments as tenant-scoped records. Framework requirements remain external references; AchieveX stores the organization mapping and assurance state rather than claiming certification. Evidence must be traceable, time-bound where applicable, and linked to an explicit control.

## Consequences
This enables later risk, access-review, retention, and assurance workflows while preserving separation between management assertions and independent audit conclusions.
