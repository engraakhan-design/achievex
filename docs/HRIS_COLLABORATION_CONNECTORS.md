# HRIS & Collaboration Connectors

Sprint 14.4 adds governed provider adapters for Workday, SAP SuccessFactors, BambooHR, HiBob, Slack, Microsoft Teams, Google Workspace and Microsoft 365. Provider payloads are normalized before entering AchieveX. Preview mode is side-effect free. Collaboration delivery records are idempotent and queued; the source does not claim live delivery without provider clients and credentials.
